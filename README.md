# Keyboard-Nilou  (KB-Nilou)
 
## 描述  
### 目前实现的功能：  
* usb键盘模拟  
* 系统音量调节  
* 上位机通讯(java kb-nilou-SDK)  
* 语音播放  
* rgb灯效  
* 热插拔键盘座  
* 震动反馈  

### 待填的坑：
* 第7K的扩展(懒得做了)  
* 官方的上位机软件(懒得写了)
